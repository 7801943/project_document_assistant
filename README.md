简介：
这是一个工程项目文件助手，主要目的是借助强大的AI模型，方便查询、读取，比较、分析工程项目文件。主要功能如下：
1. 文件查询，包括项目文件查询和规范文件查询
2. 文件读取，包括预览和ai读取，借助kkfileview，可以预览pdf,docx，xlsx等常见的办公文件。
3. 文件编辑，使用onlyoffice对office类文档进行编辑。（如果有时间的话会扩展为多人在线编辑）
4. 文件比对，使用diff函数对项目文件的差异进行比对。
5. 知识库， 使用向量检索进行知识库查询，当前（2025-7-18）仍需以来dify的知识库功能，未来有时间会使用独立pipeline实现。

技术栈：
前端 纯js+html
后端 fastapi+fastmcp
数据库 sqlite
文档预览 kkfileveiw
文档编辑 onlyoffice
